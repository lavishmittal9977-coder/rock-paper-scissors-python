# Rock Paper Scissors Game

A simple Rock Paper Scissors game built using Python.

## Features
- Play against the computer
- Best of 5 rounds
- Score tracking
- Input validation
- Exit option

## Technologies Used
- Python
- Random Module

## How to Run

```bash
python rock_paper1.py
```
